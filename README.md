### ğŸ”‘ Important Configuration (Contact Form)

The contact form uses **Formspree** to send emails. For the form to work correctly, you **must** update the action URL:

1.  Go to [Formspree](https://formspree.io/) and create an account/new form.
2.  Get your unique endpoint (e.g., `/f/xyzgajdr`).
3.  Open `index.html` and replace the placeholder in the contact form section:

    ```html
    <form action="YOUR_FORMSPREE_ENDPOINT_HERE" method="POST">
    ```
